Es6-test-sample

A sample project for es6 testing with webpack babel work flow

View project on GitHub

Sample for modern es6 code testing

using:

  • babel
  • mocha
  • webpack
  • karma
  • istanbul
  • coveralls

Usage

npm install
make test-node
make test-webpack
make test-webpack-dev
make test-karma

to get the test results
